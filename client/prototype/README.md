# Flim Visual Prototype

Open `index.html` in a browser to review the Phase 1B visual prototype.

This folder is static on purpose:

- No dependencies.
- No API calls.
- No external services.
- No movie search.
- No real movie catalog.
- No authentication.
- No database.
- No roulette logic.

The small JavaScript file only supports clickable visual route switching and placeholder poster rendering.
