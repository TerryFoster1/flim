# TV Support Roadmap

Flim now supports adding movies and TV shows to the same playlists at a lightweight title level.

## Current Scope

- Search movies, TV shows, or both through TMDb.
- Store playlist item `media_type` as `movie` or `tv`.
- Show TV badges on playlist cards.
- Add TV shows to playlists.
- Show basic TV detail pages.
- Include TV shows in Now Playing.

## Not Built Yet

- Episode tracking.
- Season progress.
- Continue watching by episode.
- New season alerts.
- Release notifications.
- TV-specific Now Playing modes.
- Provider availability by TV season or episode.

## Future Phases

1. Add season and episode models.
2. Track current season and episode.
3. Add continue-watching rows.
4. Add next-episode actions.
5. Add release and new-season notifications.
6. Add TV-specific Now Playing modes such as TV Night, Short Episodes, and Start A New Show.
