# Flim Documentation

This folder captures the product and architecture intent for Flim before implementation begins.

Documents:

- `product-bible.md` - Product vision, users, use cases, and constraints.
- `roadmap.md` - Phase-based build plan.
- `database-design.md` - PostgreSQL entity placeholders.
- `api-design.md` - API-first route and contract placeholders.
- `future-features.md` - Explicit future feature parking lot.
- `roulette-system.md` - Movie Roulette and Blind Spin architecture planning.
- `streaming-provider-strategy.md` - Provider availability, logos, links, and country-specific planning.
- `playlist-sharing-system.md` - Sharing, saving, cloning, and collaboration planning.
- `social-features.md` - Social feature boundaries and future signals.
- `deployment-notes.md` - Canonical domain and future deployment placeholders for `https://flim.ca`.
