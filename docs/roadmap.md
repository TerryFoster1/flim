# Roadmap

## Phase 1: Personal Movie Lists

Placeholder goal: allow users to save movies into poster-first personal playlists with names, descriptions, and watched status.

## Phase 1A: Product Repositioning And Architecture Update

Placeholder goal: align the scaffold around Spotify playlists for movies, poster-first browsing, streaming provider planning, playlist social behaviors, and Movie Roulette.

Implementation status: scaffold and architecture only.

## Phase 2: Playlist Sharing

Placeholder goal: allow playlist links and basic view access for friends and family.

Planning surfaces: private/shared/public visibility, share links, save playlist, and clone playlist.

## Phase 3: Collaboration

Placeholder goal: allow invited collaborators to add movies, update statuses, and coordinate shared lists.

Planning surfaces: playlist collaborators, recommended-by attribution, and shared family or friend collections.

## Phase 4: Discovery

Placeholder goal: allow browsing and following public playlists through poster-first shelves and public playlist cards.

Planning surfaces: public playlist discovery, playlist follower counts, and popularity signals.

## Phase 5: Recommendation System

Placeholder goal: introduce recommendation logic after enough product behavior and data contracts exist.

Planning surfaces: recommendation badges, recommendation attribution, and future engine output contracts.

## Phase 6: Advanced Social Features

Placeholder goal: explore richer social layers only after the core playlist product is stable.

Planning surfaces: playlist popularity, who recommended a movie, discussion/comments, and watch parties.
